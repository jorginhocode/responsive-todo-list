# Responsive To Do List
## 📌 Project Features
- Built using HTML, CSS & JS.
- Fully responsive with a clean and modern UI.
- Built to practice and improve JavaScript skills.
## 📦 Download the project directly
1. Click the green **code** button.
2. Click **Download ZIP**.
3. Extract the ZIP file and open the project in your code editor.
## 🔗 Clone or Fork the repository
You can **clone** the repository to your local machine or **fork** it in your GitHub account to get all the project's resources.
```bash
# Clone the repository using HTTPS
git clone https://github.com/jorginhocode/responsive-to-do-list.git
```
